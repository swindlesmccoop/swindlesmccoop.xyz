SSMM is a new project that I'm working on that is a Simple Script Mod Manager. It will only use GNU CoreUtils and should run on any bash shell on any machine. It is currently being written as a Fish script, but when I initialize the git repository, it will be written to work with pure bash with hopefully no prerequisites. It only uses grep, cat, echo, cp, mv, and other commands from the Unix days.
